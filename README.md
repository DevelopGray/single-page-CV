# Stephen Gray - One-Page CV Website

This repository contains the source code for my single-page CV website, a concise and interactive digital resume showcasing my experience, skills, and education. This project follows best practices for web-based CVs, inspired by the guidelines from [roadmap.sh](https://roadmap.sh/projects/single-page-cv).

## About the Project

The single-page CV format is designed to provide a streamlined and visually appealing overview of my professional background, including:
- **Skills**: Highlights key technical skills relevant to web development.
- **Education**: Brief background on relevant educational experiences.
- **Experience**: Key achievements and responsibilities from my time in the U.S. Army and other roles.
- **Portfolio Links**: Quick access to my LinkedIn and GitHub profiles for further exploration of my work.

## Purpose

This project demonstrates a practical approach to building a digital CV, providing:
- **Simple Navigation**: All essential information is presented on one page for easy reading.
- **Responsive Design**: The layout adapts for both desktop and mobile viewing.
- **Semantic HTML Structure**: Sections are organized with meaningful HTML tags to enhance accessibility, improve SEO, and allow for easy updates and customization.

## Resources

The project was structured according to the single-page CV roadmap from [roadmap.sh](https://roadmap.sh/projects/single-page-cv).

## How to Use

To view the CV website, simply open the `index.html` file in a web browser. This page can also be deployed to any static hosting service (e.g., GitHub Pages) for online accessibility.
